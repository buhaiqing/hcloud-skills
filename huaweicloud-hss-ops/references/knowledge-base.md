# HSS Fault Knowledge Base

> 本知识库汇总企业主机安全(HSS)常见故障模式、根因定位速查与跨 skill 联动。HSS 无独立 `monitoring.md`,信号提取自 `SKILL.md` 与 AIOps 模式。命令示例以 `hcloud` CLI 为主,不确定参数以 `{{user.*}}` / `{{env.*}}` 占位。

---

## 常见故障模式

| 故障ID | 症状 | 根因 | 排查步骤 | 恢复动作 |
|---|---|---|---|---|
| HSS-01 | 告警风暴 | 单日 `ListEvents > 50` 条或环比 `> 200%` | 聚类告警,识别共性源 | 聚合处理;定位攻击源 |
| HSS-02 | 未处理告警堆积 | 未处理告警连续 3 天增长 | 查未处理队列趋势 | 清理/转工单;复盘 SLA |
| HSS-03 | 漏洞修复率低 | 修复率 `< 80%` | 查未修复漏洞清单 | 批量修复;例外审批 |
| HSS-04 | 主机健康分低 | Agent offline / 未防护 | 查主机防护状态与 Agent 心跳 | 重装/启动 Agent;开启防护 |
| HSS-05 | 恶意软件 | 检测到木马/勒索软件 | 查恶意文件与进程 | 隔离主机;清除并溯源 |
| HSS-06 | 暴力破解 | 同 IP `login_fail >= 10` | 查登录失败源 IP | 封禁 IP;强化口令/MFA |
| HSS-07 | 网页防篡改失效 | 文件被篡改未拦截 | 查防护目录与策略 | 恢复文件;重开防篡改 |
| HSS-08 | 基线不合规 | 配置偏离安全基线 | 查基线检查项 | 按基线加固 |

---

## 根因定位速查

- **告警风暴** → 先判定是真实攻击还是配置抖动;同 IP 高频失败指向暴力破解。
- **Agent offline** → 多为实例停机或 Agent 进程异常,优先重装/启动而非重装系统。
- **漏洞修复率低** → 区分"已忽略例外"与"逾期未修",后者需升级处理。
- **暴力破解** → 封禁源 IP 仅为缓解,须同步强化认证(口令复杂度/MFA)。
